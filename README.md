# Combination-Lock (Arduino Projects)

Push Button as Combination key to unlock the solenoid

List of components:
i) Arduino Uno
ii) Led
iii) Jumper Wire
iv) Push button
v) Breadboard
vi) Solenoid Lock
vii) L298N
